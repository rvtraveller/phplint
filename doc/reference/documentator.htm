Documentator

PHPLint can generate an HTML document for any package just parsed. This HTML document contains:

  • Version of PHP.
  • Required PHP modules.
  • Required packages.
  • All the constants, variables, functions, classes and their members (items, for short).
  • For each item, its DocBlock with types and in-line tags resolved.

All the references to other modules and packages are made as relative paths, so that all the documents generated can be moved without breaking the references.

To enable the generation of the HTML document, the --doc command line option must be given before the name of the package to be parsed. Example: 

C:\phplint>phpl --doc MyProgram.php

will generate the file MyProgram.html.

The --doc-help command line option displays all the available options of the PHPLint Documentator: 

C:\phplint>phpl --doc-help
PHPLint Documentator

Options:
  --doc                 enable documentation generator
  --doc-extension EXT   extension of the generated documents (def. ".html")
  --doc-page-header H   header text of the generated HTML page
  --doc-page-footer F   footer text of the generated HTML page
  --doc-ref-remap A B   remap HTML anchors references from A* to B*
